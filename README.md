# Hardening de Base de Datos e Indicadores de Seguridad

Este repositorio contiene una propuesta técnica para asegurar la base de datos de un comercio electrónico y establecer métricas de seguimiento de seguridad. Fue desarrollado para la materia de **Monitoreo y Gestión de la Seguridad** en el programa de **Técnico en Servicios de Seguridad Informática**.

## Contenido de la propuesta
* **Seguridad en PostgreSQL**: Aplicación de control de accesos, encriptación y el principio de privilegios mínimos.
* **Indicadores clave de rendimiento**: Propuesta matemática de 5 indicadores de seguridad para rastrear intentos de accesos fallidos, éxito de copias de seguridad, anomalías de extracción, y tiempos de instalación de parches.
* **Automatización**: Scripts utilitarios escritos en Python para procesar los textos de los reportes.

## Archivos del repositorio
* `docs/`: Propuesta en PDF legible con las fórmulas matemáticas y el libro de indicadores.
* `scripts/`: Scripts complementarios en Python.
* `latex/`: Archivo de maquetación en LaTeX y recurso gráfico del caso.
